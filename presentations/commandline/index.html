Handling Files

Handling Files

  				$ cat <file>                   # print file
  				$ touch <file>                 # create file
  				$ mv <file> <destination>      # move file
  				$ cp <file> <destination>      # copy file
  				$ rm <file>                    # remove file

Handling Directories

  				$ mkdir <direcotry>                 # make directory
  				$ mv <directory> <destination>      # move directory
  				$ cp -r <directory> <destination>   # copy directory
  				$ rm -r <direcotry>                 # remove directory

Searching

Searching for files

  				$ find <path> <expression>

  				# All files ending with .txt
  				$ find . -name "*.txt"

  				# All files ending with .txt
  				# and changed in the last 10 minutes
  				$ find . -name "*.txt" -ctime -10m

Searching inside files

  				$ grep <pattern> <filename>

					# Search for "ots" in file foo.txt
  				$ grep "ots" foo.txt   

					# Recursively search for "ots"
					# in all files in the current directory
  				$ grep "ots" -r .

IO Redirect and Pipes

Redirecting Input and Output

  				# Write directory list into file
  				$ ls > content.txt

  				# Write the first ten lines into file
  				$ head book-1.txt > preview.txt

  				# Concaternate files and write into single file
  				$ cat book-1.txt book-2.txt > book-full.txt

Chaining Commands with Pipes

  				# Search for the word "Flint"
  				$ grep "Flint" book-1.txt | less 

  				# Count how often "Flint" is written in the book
  				$ cat book-1.txt book-2.txt | grep "Flint" | wc -l 

  				# Extract the paths from a log file
  				$ cat nasa.log | awk '{ print $7 }' | less
