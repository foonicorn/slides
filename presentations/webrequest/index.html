The Request

Minimal Request

					GET /index.html HTTP/1.1
					Host: learn.opentechschool.org

Method

					GET /index.html HTTP/1.1
					Host: learn.opentechschool.org

Resource

					GET /index.html HTTP/1.1
					Host: learn.opentechschool.org

Version

					GET /index.html HTTP/1.1
					Host: learn.opentechschool.org

Host

					GET /index.html HTTP/1.1
					Host: learn.opentechschool.org

Example Request

					GET /index.html HTTP/1.1
					Host: learn.opentechschool.org
					User-Agent: Mozilla/5.0; Macintosh; Intel Mac OS X 10.9
					Accept: text/html,application/xhtml+xml,application/xml
					Accept-Language: en-US,en;q=0.5
					Accept-Encoding: gzip, deflate
					DNT: 1
					Connection: keep-alive

The Response

Minimal Response

					HTTP/1.1 200 OK

Version

					HTTP/1.1 200 OK

Status Code

					HTTP/1.1 200 OK

  • 1xx: Informational
  • 2xx: Success
  • 3xx: Redirection
  • 4xx: Client Error
  • 5xx: Server Error

Example Response

					HTTP/1.1 200 OK
Server: GitHub.com
Content-Type: text/html; charset=utf-8
Expires: Tue, 04 Mar 2014 21:59:43 GMT

Cookies

Set in the response

					HTTP/1.1 200 OK
					Set-Cookie: ID=31d4d96e407aad42;

Sent with every request

					GET /index.html HTTP/1.1
					Host: learn.opentechschool.org
					Cookie: ID=31d4d96e407aad42

Cookie Parameters

					Set-Cookie: ID=31d4d96e407aad42;
					            Expires=Wed, 09 Jun 2021 10:18:14 GMT;
					            Max-Age=3600;
					            Path=/;
					            Secure;
					            HttpOnly
